Frost Particle Meshing for Blender

Installation:
1. Open Blender 5+
2. Go to Edit > Preferences > Add-ons
3. Open the add-on menu and choose Install from Disk...
4. Select the release zip from GitHub Releases
5. Enable "Frost Particle Meshing"

Notes:
- This package uses frost_native.dll through native_bridge.py.
- Restart Blender after replacing the addon files or updating the native DLL.
- The current public package targets the CPU path and the experimental Vulkan backend in one addon.
